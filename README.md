# java-global-learning-journey

This repository contains assignments and practical implementations completed during the EPAM Java training program.

The purpose of this repository is to demonstrate understanding of core Java and Spring technologies through modular projects developed during the training.

## Topics Covered

The training program includes the following topics:

* Java 8+ Functional Programming and Stream API
* Spring Core
* Spring Data Access
* REST API – Architecture
* Monitoring and Troubleshooting Java Applications

## Repository Structure

Each module of the training is implemented as a separate project directory.

```
java-global-learning-journey
│
├── java-functional-programming
├── spring-core
├── spring-data-access
├── rest-api
├── monitoring-troubleshooting
│
└── README.md
```

Each module contains:

* Source code
* Configuration files
* Unit tests (where applicable)
* Implementation of the required tasks for that module

## Technologies Used

* Java 8+
* Maven
* Spring Framework
* Hibernate / JPA
* REST APIs
* Logging frameworks

## Purpose

This repository serves as a collection of completed training assignments and demonstrates the application of modern Java development practices, including:

* Clean code principles
* Modular project structure
* Dependency injection
* Data persistence
* RESTful service design
* Application monitoring and troubleshooting techniques
